# Todo List App

A modular, project-based Todo List application built with Vanilla JavaScript.
This app allows users to create projects, manage todos within each project, and persist data using localStorage.

# Features

- Create and delete projects
- Default project on first load
- Add todos to specific projects
- Edit existing todos
- Expand todo details
- Delete todos
- Assign due dates and priority levels
- Persistent storage using localStorage
- Clean modular architecture (separated logic & DOM)
- Date formatting using date-fns

# Tech Stack

- HTML
- CSS
- Vanilla JavaScript (ES6 Modules)
- Webpack
- date-fns
- Web Storage API (localStorage)

# Future Improvements
